# Vitrine ML5 - Global Context

## Projeto
Este é o repositório central da Vitrine ML5, um ecossistema de sub-projetos educacionais e experimentais.
- **URL Local:** http://localhost:3000/vitrine-ml5/
- **Idioma Principal:** Português (PT-BR) para documentação e interface.
- **Estilo de Código:** Vanilla CSS, HTML semântico, JavaScript moderno (ES6+). Evitar TailwindCSS a menos que solicitado.

## Diretrizes de Contexto
1. **Hierarquia:** Sempre verifique se existe um `GEMINI.md` dentro da subpasta do projeto antes de realizar alterações profundas. As instruções locais têm precedência sobre estas globais.
2. **Navegação:** Ao trabalhar em uma subpasta (ex: `mlab5/`), foque nos padrões encontrados nela.
3. **Imagens:** Muitos projetos usam placeholders. Mantenha a consistência visual retro-futurista/educacional.
