O HTML é a linguagem para construção de páginas web.

Usar o HTML para montar uma página web é como decorar uma casa. Cada móvel tem uma função: Cabe a você definir onde ficará a TV, os quadros as fotos no porta retratos, estante, livros etc.

O estilo, a princípio, fica por nossa conta.

Quando você terminar de aprender o HTML, aí sim você vai poder fazer o curso de CSS3 para aprender a escolher as cores, estilos e enfeites do seu jeito.
