✨ CSS3 - Transições e Animações

Transições CSS3

As transições CSS3 permitem criar mudanças suaves entre estados de elementos, melhorando significativamente a experiência do usuário com animações fluidas e naturais.

Vantagens das Transições:

  • Mudanças suaves entre estados
  • Melhor experiência do usuário
  • Performance otimizada pelo navegador
  • Fácil de implementar
  • Suporte amplo nos navegadores
  • Reduz necessidade de JavaScript

Propriedade transition

A propriedade transition é a base das transições CSS, definindo como as mudanças de propriedades devem ser animadas.

Propriedades da Transition:

  • transition-property - Propriedades a animar
  • transition-duration - Duração da animação
  • transition-timing-function - Função de timing
  • transition-delay - Atraso para iniciar
  • transition - Shorthand das propriedades

Exemplos de Transição:

/* Transição básica */
.botao {
  background-color: #007bff;
  transition: background-color 0.3s ease;
}

.botao:hover {
  background-color: #0056b3;
}

/* Transição múltipla */
.card {
  transition: all 0.3s ease-in-out;
  transform: scale(1);
}

.card:hover {
  transform: scale(1.05);
  box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}

/* Transição específica */
.elemento {
  transition: opacity 0.5s ease 0.2s;
}

Funções de Timing

As funções de timing controlam a velocidade da animação ao longo do tempo, criando diferentes sensações de movimento.

Funções Disponíveis:

  • ease - Lento, rápido, lento (padrão)
  • linear - Velocidade constante
  • ease-in - Lento no início
  • ease-out - Lento no fim
  • ease-in-out - Lento no início e fim
  • cubic-bezier() - Função personalizada

Exemplos de Timing:

/* Diferentes sensações */
.rapido { transition: all 0.2s ease; }
.normal { transition: all 0.3s ease-in-out; }
.lento { transition: all 0.8s ease-out; }

/* Função personalizada */
.personalizado {
  transition: all 0.5s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

/* Transições específicas */
.elemento {
  transition: 
    width 0.3s ease-in,
    height 0.3s ease-out,
    opacity 0.2s linear;
}

Animações com @keyframes

As animações CSS3 permitem criar movimentos complexos e contínuos usando a regra @keyframes para definir os estágios da animação.

Conceitos das Animações:

  • @keyframes - Define estágios da animação
  • from/to ou porcentagens - Pontos da animação
  • animation-name - Nome da animação
  • animation-duration - Duração total
  • animation-iteration-count - Número de repetições
  • animation-direction - Direção da animação

Criando Keyframes:

/* Animação básica */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

/* Animação com múltiplos estágios */
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-20px); }
}

/* Animação complexa */
@keyframes slideInLeft {
  0% {
    transform: translateX(-100%);
    opacity: 0;
  }
  50% {
    transform: translateX(10%);
  }
  100% {
    transform: translateX(0);
    opacity: 1;
  }
}

Propriedades de Animation

As propriedades de animation controlam como as animações @keyframes são executadas, incluindo duração, repetição e direção.

Propriedades Principais:

  • animation-name - Nome da animação
  • animation-duration - Duração em segundos
  • animation-timing-function - Função de timing
  • animation-delay - Atraso para iniciar
  • animation-iteration-count - Número de repetições
  • animation-direction - Direção (normal/reverse/alternate)
  • animation-fill-mode - Estado antes/depois
  • animation-play-state - Pausar/executar

Aplicando Animações:

/* Aplicação básica */
.elemento {
  animation: fadeIn 1s ease-in-out;
}

/* Configuração detalhada */
.botao {
  animation-name: pulse;
  animation-duration: 2s;
  animation-timing-function: ease-in-out;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

/* Shorthand */
.loading {
  animation: spin 1s linear infinite;
}

/* Múltiplas animações */
.complexo {
  animation: 
    fadeIn 0.5s ease-in,
    slideIn 0.8s ease-out 0.3s;
}

Estados de Animação

animation-fill-mode e animation-play-state permitem controlar o estado dos elementos antes, durante e após as animações.

animation-fill-mode:

  • none - Não aplica estilos (padrão)
  • forwards - Mantém estado final
  • backwards - Aplica estado inicial
  • both - Aplica inicial e final

Exemplos de Estados:

/* Mantém estado final */
.fade-in {
  animation: fadeIn 1s ease-in;
  animation-fill-mode: forwards;
}

@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

/* Controle de execução */
.pausavel {
  animation: rotate 2s linear infinite;
}

.pausavel:hover {
  animation-play-state: paused;
}

/* Estado inicial aplicado */
.slide-in {
  animation: slideIn 0.5s ease-out;
  animation-fill-mode: backwards;
}

Animações de Performance

Para animações suaves e performáticas, é importante usar propriedades que não causem repaint ou reflow do navegador.

Propriedades Performáticas:

  • transform - Muda posição/rotação sem reflow
  • opacity - Muda transparência suavemente
  • filter - Efeitos visuais otimizados
  • will-change - Dica para otimização
  • Evitar: width, height, margin, padding

Otimização de Performance:

/* ✅ Performático */
.elemento {
  transform: translateX(100px);
  opacity: 0.5;
  will-change: transform, opacity;
}

/* ❌ Evitar */
.elemento-lento {
  width: 200px;
  height: 100px;
  margin-left: 50px;
}

/* Hardware acceleration */
.acelerado {
  transform: translateZ(0);
  /* ou */
  transform: translate3d(0, 0, 0);
}

/* Exemplo otimizado */
.card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.card:hover {
  transform: translateY(-5px);
  box-shadow: 0 10px 20px rgba(0,0,0,0.2);
}

Casos de Uso Práticos

Transições e animações são fundamentais para criar interfaces modernas e interativas que guiam a atenção do usuário.

Aplicações Comuns:

  • Hover effects em botões e cards
  • Loading spinners e indicadores
  • Transições de página
  • Micro-interações
  • Feedback visual de ações
  • Animações de entrada/saída

Exemplos Práticos:

/* Botão com hover suave */
.btn {
  background: linear-gradient(45deg, #667eea, #764ba2);
  transition: all 0.3s ease;
  transform: scale(1);
}

.btn:hover {
  transform: scale(1.05);
  box-shadow: 0 5px 15px rgba(102, 126, 234, 0.4);
}

/* Loading spinner */
@keyframes spin {
  to { transform: rotate(360deg); }
}

.spinner {
  animation: spin 1s linear infinite;
}

/* Card com efeito de entrada */
@keyframes slideInUp {
  from {
    transform: translateY(30px);
    opacity: 0;
  }
  to {
    transform: translateY(0);
    opacity: 1;
  }
}

.card {
  animation: slideInUp 0.6s ease-out;
}

  • Keyframes = Animações complexas com múltiplos estágios
  • Performance = Use transform e opacity para otimização
  • Timing = Diferentes funções de easing para variadas sensações
