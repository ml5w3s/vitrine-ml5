✨ Efeitos e Animações

Efeitos jQuery

jQuery oferece uma variedade de efeitos visuais que tornam as interfaces mais dinâmicas e atraentes. Os efeitos são divididos em categorias principais: fade, slide, show/hide e animações customizadas.

Categorias de efeitos:

  • Efeitos Fade: fadeIn, fadeOut, fadeToggle, fadeTo
  • Efeitos Slide: slideUp, slideDown, slideToggle
  • Show/Hide: show, hide, toggle
  • Animações: animate (customizado)

Efeitos Fade

Os efeitos fade controlam a opacidade dos elementos, criando transições suaves de aparecer e desaparecer.

Métodos fade:

  • fadeIn(): Aparece gradualmente
  • fadeOut(): Desaparece gradualmente
  • fadeToggle(): Alterna entre aparecer/desaparecer
  • fadeTo(): Define opacidade específica

Exemplos de fade:

// ✅ Efeitos fade básicos
$("#elemento").fadeIn();           // Aparece com duração padrão
$("#elemento").fadeOut();          // Desaparece com duração padrão
$("#elemento").fadeToggle();       // Alterna entre aparecer/desaparecer

// ✅ Fade com duração específica
$("#elemento").fadeIn(1000);       // Aparece em 1 segundo
$("#elemento").fadeOut("slow");    // Desaparece lentamente
$("#elemento").fadeToggle("fast"); // Alterna rapidamente

// ✅ fadeTo - define opacidade específica
$("#elemento").fadeTo(1000, 0.5);  // 50% de opacidade em 1 segundo
$("#elemento").fadeTo("slow", 0);  // Totalmente transparente

// ✅ Com callbacks
$("#elemento").fadeIn(1000, function() {
    alert("Elemento apareceu!");
});

$("#elemento").fadeOut(500, function() {
    $(this).remove(); // Remove elemento após desaparecer
});

// ✅ Exemplo prático
$(document).ready(function() {
    $("#fadeIn").click(function() {
        $("#imagem").fadeIn("slow");
    });
    
    $("#fadeOut").click(function() {
        $("#imagem").fadeOut("slow");
    });
    
    $("#fadeToggle").click(function() {
        $("#imagem").fadeToggle(1000);
    });
});

Efeitos Slide

Os efeitos slide animam a altura dos elementos, criando movimento de deslizar para cima ou para baixo.

Métodos slide:

  • slideUp(): Desliza para cima (oculta)
  • slideDown(): Desliza para baixo (mostra)
  • slideToggle(): Alterna entre slideUp/slideDown

Exemplos de slide:

// ✅ Efeitos slide básicos
$("#elemento").slideUp();          // Desliza para cima
$("#elemento").slideDown();        // Desliza para baixo
$("#elemento").slideToggle();      // Alterna entre os dois

// ✅ Slide com duração
$("#elemento").slideUp(800);       // Desliza para cima em 800ms
$("#elemento").slideDown("slow");  // Desliza para baixo lentamente
$("#elemento").slideToggle("fast"); // Alterna rapidamente

// ✅ Com easing (suavização)
$("#elemento").slideUp(1000, "swing");   // Movimento suave
$("#elemento").slideDown(800, "linear"); // Movimento linear

// ✅ Com callbacks
$("#elemento").slideUp(500, function() {
    console.log("Elemento deslizou para cima");
});

$("#elemento").slideDown(1000, function() {
    $(this).css("background-color", "yellow");
});

// ✅ Exemplo prático - menu acordeão
$(document).ready(function() {
    $(".menu-titulo").click(function() {
        $(".menu-conteudo").slideToggle("slow");
    });
    
    // Fechar todos os menus
    $("#fechar-todos").click(function() {
        $(".menu-conteudo").slideUp("fast");
    });
    
    // Abrir todos os menus
    $("#abrir-todos").click(function() {
        $(".menu-conteudo").slideDown("fast");
    });
});

Show, Hide e Toggle

Estes métodos controlam a visibilidade dos elementos de forma instantânea ou com efeitos personalizados.

Métodos básicos:

  • show(): Mostra o elemento
  • hide(): Oculta o elemento
  • toggle(): Alterna entre mostrar/ocultar

Exemplos de show/hide:

// ✅ Métodos básicos
$("#elemento").show();             // Mostra instantaneamente
$("#elemento").hide();             // Oculta instantaneamente
$("#elemento").toggle();           // Alterna instantaneamente

// ✅ Com duração (usa fade)
$("#elemento").show(1000);         // Aparece em 1 segundo
$("#elemento").hide("slow");       // Desaparece lentamente
$("#elemento").toggle("fast");     // Alterna rapidamente

// ✅ Com efeito específico
$("#elemento").show("slide");      // Mostra com slide
$("#elemento").hide("fade");       // Oculta com fade

// ✅ Com callbacks
$("#elemento").show(500, function() {
    alert("Elemento mostrado!");
});

$("#elemento").hide(300, function() {
    $(this).css("display", "none");
});

// ✅ Exemplo prático - painel de controle
$(document).ready(function() {
    $("#btn-mostrar").click(function() {
        $("#painel").show("slide", 500);
    });
    
    $("#btn-ocultar").click(function() {
        $("#painel").hide("fade", 300);
    });
    
    $("#btn-alternar").click(function() {
        $("#painel").toggle("slow");
    });
});

Animações Customizadas

O método animate() permite criar animações personalizadas alterando propriedades CSS ao longo do tempo.

Características do animate():

  • Propriedades CSS: width, height, opacity, margin, etc.
  • Duração: Tempo da animação em milissegundos
  • Easing: Tipo de suavização (swing, linear)
  • Callback: Função executada ao final

Exemplos de animate():

// ✅ Animações básicas
$("#elemento").animate({
    width: "200px",
    height: "100px"
});

$("#elemento").animate({
    opacity: 0.5,
    marginLeft: "50px"
}, 1000);

// ✅ Animações com duração
$("#elemento").animate({
    fontSize: "24px",
    backgroundColor: "#ff0000"
}, "slow");

// ✅ Animações em sequência
$("#elemento")
    .animate({width: "300px"}, 500)
    .animate({height: "200px"}, 500)
    .animate({opacity: 0.5}, 300);

// ✅ Animações com easing
$("#elemento").animate({
    left: "500px"
}, {
    duration: 1000,
    easing: "swing"
});

$("#elemento").animate({
    top: "200px"
}, {
    duration: 800,
    easing: "linear"
});

// ✅ Animações com callback
$("#elemento").animate({
    width: "400px"
}, 1000, function() {
    $(this).animate({
        height: "300px"
    }, 500);
});

// ✅ Exemplo prático - expansão de card
$(document).ready(function() {
    $(".card").hover(
        function() {
            $(this).animate({
                width: "350px",
                height: "250px",
                marginTop: "-10px"
            }, 300);
        },
        function() {
            $(this).animate({
                width: "300px",
                height: "200px",
                marginTop: "0px"
            }, 300);
        }
    );
});
