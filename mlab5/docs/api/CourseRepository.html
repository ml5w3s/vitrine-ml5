Class: CourseRepository

CourseRepository()

CourseRepository - Singleton para gerenciar a lógica de negócios e a transformação de dados de cursos. Este repositório é responsável por consumir dados brutos do ApiService e transformá-los em instâncias de modelos (Course, Lesson), aplicando a lógica de negócios necessária.
