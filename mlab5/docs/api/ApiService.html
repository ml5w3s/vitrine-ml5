Class: ApiService

ApiService()

ApiService - Singleton para gerenciar a comunicação com a fonte de dados (arquivos JSON). Este serviço é responsável APENAS por buscar (fetch) os dados brutos. A transformação e a lógica de negócios são tratadas pelo CourseRepository.
